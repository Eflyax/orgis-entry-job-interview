# Orgis - Entry job interview

## Project Setup

```sh
docker-compose up -d
```

### Exec docker container

```sh
docker exec -ti orgis-fe bash
```

### Run dev build

```sh
yarn dev
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
yarn test:unit
```
### Run production build

```sh
yarn build
```
